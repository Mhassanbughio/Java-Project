### Java
I used java 1.8.0_241 
### How To Use
You can **rename/delete** the package line and add the files to your folder. Run the **Main.java**
### Tools
I'm using **Java.Util.Random** for the dice and the output is integer between 1-12 and **Java.util.Scanner** to read the input from keyboard
### input
The first line is **board size** (ex 100) (**int**)  
The next line is the n **number of player** (ex 5) (**int**)  
and there will be n line to add **player's name** (ex bejo) (**string**)
